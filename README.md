# 🎓 Campus Connect

Campus Connect is a campus-focused collaboration platform that connects students based on skills, requirements, and interests.  
It enables students to collaborate on projects, complete gigs to earn pocket money, and build a data-driven **Placement Readiness Profile** based on real work and peer ratings.

---

## 🚀 Features

- Skill-based student discovery within campus
- Gig posting and mutual acceptance workflow
- In-app communication after gig acceptance
- Peer rating and feedback system
- AI-generated Placement Readiness Card
- Collaboration for projects, learning, and freelancing
- Secure and campus-verified user access

---

## 🧠 Placement Readiness Card

After completing gigs, students receive ratings that are processed to generate a **Placement Readiness Card**, which includes:
- Readiness percentage score
- Highlighted skills
- Number of completed gigs
- Performance-based insights

This provides a transparent and practical indicator of industry readiness.
